# Open Source Starter (ZH)

> 一套中文开源项目模板：README、贡献指南、Issue/PR 模板、CI、Release，一键把你的仓库做得专业可协作。

如果对你有帮助，欢迎点个 Star ⭐
**最新版本：** [v0.1.0](../../releases/tag/v0.1.0)
## ✨ 你将获得
- ✅ 专业 README 模板（徽章 / 目录 / 示例 / Roadmap）
- ✅ LICENSE / CHANGELOG / 安全说明
- ✅ CONTRIBUTING + Code of Conduct（更像“真实开源项目”）
- ✅ Issue 模板（Bug / Feature）+ PR 模板
- ✅ GitHub Actions CI（自动检查）
- ✅ docs：从 0 到 1 的中文开源维护指南

## 🚀 快速开始
1. 点击右上角 **Use this template** 创建你的新仓库
2. 按需修改 `README.md` 与 `docs/` 文档
3. 开始接收 Issue / PR，持续迭代
