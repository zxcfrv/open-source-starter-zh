# 贡献指南（CONTRIBUTING）

感谢你愿意贡献！你可以通过以下方式参与：
- 提交 Bug：请使用 Issue 模板
- 提交需求：请使用 Feature 模板
- 提交 PR：请按 PR 模板填写背景、改动与验证方式

## 开发约定
- 提交信息建议格式：`type: message`
  - type: feat / fix / docs / chore / refactor
- 尽量在一个 PR 里只做一件事
- 重要改动请补充到 CHANGELOG.md

## 提交流程
1. Fork 本仓库
2. 新建分支：`feat/xxx` 或 `fix/xxx`
3. 提交改动并推送
4. 发起 Pull Request，并按模板填写
